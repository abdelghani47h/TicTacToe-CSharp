## 🎯 Gameplay Features

- 👥 **Two Players** — Alternate turns between Player 1 (❌ Pink) and Player 2 (🟢 Green).  
- 🧠 **Smart Win Detection** — Automatically checks for wins and draws.  
- 👁️ **Visual Indicators** — Highlights the current player clearly.  
- ⚡ **Responsive Interaction** — Smooth and instant feedback for every move.

## ⚙️ Implementation Details

### 🧩 Game Board
- 🧱 **3×3 Grid** built with PictureBox controls.  
- ⚙️ **Dynamic Initialization** using arrays for efficient handling.  
- 🧠 **State Tracking** with Tag properties to manage moves.  

### 🧠 Game Logic
- 🔄 **Turn Management** — Automatic switching between players.  
- 🚫 **Validation** — Prevents moves on occupied cells.  
- 🏁 **Win/Draw Detection** — Evaluates all possible combinations.  
- 🔁 **Restart Option** — Reset the board instantly.  

---

## 🧮 Game States

| State | Description |
|-------|-------------|
| 🎯 Initialization | Game setup and board reset |
| ⏳ In Progress | Active gameplay |
| 🏆 Win State | Player victory |
| 🤝 Draw State | No winner |
| 🔁 Reset Ready | Awaiting next round |

## 🛠️ Setup & Installation

### 📋 Requirements
- 💻 **.NET Framework 4.5 or higher**  
- 🪟 **Windows OS**  
- 🧩 **Visual Studio**  

### ⚙️ Steps
1. Open **Visual Studio**  
2. Navigate to **Tools → NuGet Package Manager → Manage NuGet Packages for Solution**  
3. Search for **Guna.UI2.WinForms**  
4. Install the package  
5. Rebuild the solution  

✅ Guna.UI2 is required for modern UI elements.

---

## 🚀 Running the Game

1. **Clone** the repository  
2. **Open** it in Visual Studio  
3. **Build** the solution  
4. **Run** the application 🎮
